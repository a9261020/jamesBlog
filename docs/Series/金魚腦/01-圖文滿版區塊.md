## 重點整理

- 想要用兩張圖片的話，可以使用 background:第一個圖片,第二個圖片

- 可以設定此顏色要佔比多少

```css
.banner {
  background: linear-gradient(115deg, #7bf 50%, transparent 50%) center center/100%,
    url("https://picsum.photos/1200/600?random=10") right center/100%;
}
```
