## 重點整理

- img 會因為 baseline 的問題導致圖片下方會有點空格，使用(vertical-align)去除。

```css
img {
  vertical-align: middle;
}
```

- 想要用 border 去當作長寬的話，需要先把原先的 width、height 設定為 0
